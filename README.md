# Git Lab CLI-1

A lightweight analytics CLI application for generating reports and exporting dashboard metrics.

## Features

- Dashboard metrics
- Report generation
- CSV export

## Testing

Run the export verification test:

```bash
npm test
```

## Branches

- `main` contains ongoing development.
- `release-1.0` represents the current production release.

## Export Workflow

CSV exports include:

- generation timestamp
- column headers
- metrics rows

Exports are intended for reporting and archival purposes.
